[Q1] WHAT IS CONSIDERED A 'RANDOM GUESS' IN A MULTICLASS CLASSIFICATION (100 WORDDS)

>> A RANDOM GUESS IN MULTICLASS CLASSIFICATION REFERS TO PREDICTING LABELS ENTIRELY AT RANDOM,

>> WITHOUT UTILIZING (LEARN) ANY INFORMATION FROM THE TRAINING DATA.

>> GAUHER(2016) CLAIMED THAT THE ACCURACY OF RANDOM GUESSING IN MULTICLASS CLASSIFICATION SHOULD EQUAL TO 1/K,

>> WHERE K IS THE NUMBER OF CLASSES. FROM THIS, I INFER THAT IF MY MODEL'S ACCURACY GREATER THAN 1/10 OR 10%,

>> IT IS LIKELY THAT IT PERFORMING BETTER THAN RANDOM GUESSING.

>> MOREOVER, SINCE WE ADD MORE DATA AUGMENTATION TECHNIQUES TO THE MODEL AND OBSERVING CHANGES IN ACCURACY (E.G., WITH ENSEMBLE METHODS)

>> THIS SUGGESTS THAT THE DATA PLAYS A ROLE IN THE MODEL'S PREDICTIONS.

>> REFERENCE: https://blog.revolutionanalytics.com/2016/03/classification-models.html


[Q2] JUSTIFY BRIEFLY YOUR METRICS IN A PRINTED COMMENT (50 WORDS)

>> BOTH F1-SCORE AND ACCURACY ARE REPORTED: ACCURACY IS SIMPLE AND INTUITIVE, WHILE F1-SCORE IS BETTER SUITED FOR IMBALANCED MULTICLASS DATA.

>> ALTHOUGH F1-SCORE MAY BE MORE APPROPRIATE FOR THIS CLASSIFICATION TASK, ACCURACY REMAINS USEFUL FOR CAPTURING OVERALL PERFORMANCE TRENDS.

>> NOTE THAT, ACCURACY IS USED TO COMPARE MODELS PERFORMANCE.
